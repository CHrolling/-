## 第三章 数字计算

### 数据的类型

- int:整数，可以是正数或是负数。**对int操作产生int数据。**
- float：小数，**对float操作产生float数据。**
- type:展示任何数据的数据类型
+ python内置数值操作：
    | 操作符 | 操作 |
    | - | - |
    | + | 加 |
    | - | 减 |
    | * | 浮点除 |
    | / | 指数 |
    | ** | 指数 |
    | abs() | 绝对值 |
    | // | 整数除 |
    | % | 取余 |
    
**其中，整数除`//`和浮点除`/`得出的结果是截然不同的。例如，python当中有以下两个程序：**
```python
10/3
10//3
10/5
```
**输出结果**:
```python
3.3333333333333335
3 
2.0
```
可知`/`的输出结果一定是一个浮点数`float`

### 类型转换和舍入

+ 这里首先有一个混合float和int的例子：
```python
>>> x=5.0 * 2
    print("x=",x)
```
在自然情况下，python输出结果为：
```python
x=10.0
```
在python程序当中，python会自动将int转换为浮点数float，以此避免小数部分转换时候造成的风险。
- 如果希望可以转换两种数据，可以直接参考以下的例子：
```python
>>> int(4.5)
4
>>> float(2)
2.0
>>> int(float(3.3))
3.3
>>> float(int(2))
2
```
**int无法做到四舍五入**。
**但是你可以使用内置函数`round`来进行四舍五入的计算。**
```python
>>> round(3.27)
3
>>> round(3.74)
4
```
显然，使用round会产生一个int值。当然，round值也可以指定小数点后的某位进行四舍五入，例如：
```python
>>>  pi = 3.141592653589793 
>>> round(pi, 2) 
3.14 
>>> round(pi,3) 
3.142 
```
+ 同时，`int`和`float`也可以将**数字字符串**转换为数字。这样可以很好的替代`eval`工作，防止在该输入数字的地方输入不法代码的侵入。但这样就会失去同时输入的能力。

| cos(x) | cos x | x 的余弦 |

| log(x) | ln x | x 的自然对数（以e为底）|

| exp(x) | ex | e 的x次方 |
| ceil(x) | [x] | 最小的>=x的整数 |
| floor(x) | [x] | 最大的<=x的整数 |
