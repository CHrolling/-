# CSS学习

- Cascading Style Sheets,层叠样式表
- 用于指定页面中各个元素的颜色、字体、大小、间距、边框、背景等内容，从而实现更精确的页面设计
- HTML是毛胚结构，CSS则可以完成精装
- 通常由`选择器{}`来构成,每一行都需要用英文分号来分解

## CSS的导入方式

- 内联样式：直接将样式写在HTML的style中
- 内部样式表：单独开一段HTML，标签为`<style></style>`
- 外部样式表:单独新建一个css文件，将style单独存放并用`href`链接到HTML当中。
- **通常内联>内部>外部，优先级高的会覆盖优先级低的CSS。**

## 选择器

- 允许针对特定元素或一组元素定义样式，简单来说，就是指定要对哪一段HTML进行修饰
- 通常由以下几种选择器分类：
    1. 元素选择器
    2. 类选择器
    3. ID选择器
    4. 通用选择器
    5. 子元素选择器
    6. 后代选择器
    7. 并集选择器
    8. 伪类选择器

### 选择器结构

- 元素选择器
    - `<style>元素{定义样式;}</style>`
- 类选择器
    - 在类名前面加上一个“.”
- ID选择器
    - 为某一标签添加ID，`#+ID{}`的方式来为标签添加CSS定义样式
- 通用选择器
    - `*{}`,选中全体标签并同时定义样式
- 子元素选择器
    - 选择直接父元素内部的子元素并进行定义样式，大标签内的小标签，用`父元素标签名 > .子元素标签名{}`的结构来定义样式
- 后代选择器
    - 子代和后代的区别：子代元素的子代，即是父元素的后代。
    - 用`.附带元素标签名+标签{}`的格式定义样式
- 相邻兄弟选择器（并集选择器）
    - 选择在同一级别下的元素中紧跟选中元素之后的第一个标签
- 伪类选择器
    - 用户交互实现
    - `标签名：hover{}`的结构定义样式

## 属性

- 字体：***font***，可以设置所有关于字体的样式，只要写了font就可以设置所有的字体的颜色、字形、颜色、大小等
- 文本属性：***line-hight***,行间距
- width hight：行内元素无法添加宽高样式，只有**块元素**和**行内块元素**可以添加，但是三种元素可以通过`display`来互相转换

## 盒子模型

- 布局概念
- 包含：
    1. 内容content
    2. 内边距padding
    3. 边框border
    4. 外边距margin

## 浮动

- 网页布局方法：
    1. 标准流（普通流、文档流）：网页按照元素的书写顺序依次排列
    2. 浮动
    3. 定位
    4. 自定义布局`Flexbox``Grid`

- 与行内块的区别是，浮动框间没有空缝隙，更加美观
- 相对父元素浮动
- `float`
- 特性：
    1. 脱标
    2. 一行显示，顶部对齐
    3. 具备行内块元素的特性
- 清除浮动：
    1. 在父元素style中设置`overflow:hidden`
    2. 伪元素法
