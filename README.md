# HTML 学习

- HTML 用 **标签** 来定义文本、图像、链接等。
- 标签通常由前后两个一起构成

## HTML 的结构

1. `<!DOCTYPE html>` 声明文件属性
2. `<html></html>` 根元素，包含整个文档结构
3. `<head></head>` 头部，包含文件原信息、标题、编码格式、外部样式等内容
4. `<body></body>` 对浏览器实际展示的内容
